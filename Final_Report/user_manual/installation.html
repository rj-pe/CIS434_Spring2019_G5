Installation

Steps to Download:

Our chess program can be found in the included Group_5_Chess.jar file submitted, or it can be found on our group's GitHub.

Steps to Run:

The program can either be run by double clicking the Group_5_Chess.jar file, or through the command line with command "java -jar Group_5_Chess.jar" after changing to the file's correct directory. Currently, the jar is known to run properly with Java SE 9.
